## Re-create App:

```bash
nvm use 18.12.1 || nvm install 18.12.1
npm install -g npm@9.1.2
npm install -g @ionic/cli
ionic start myApp tabs --type vue
cd myApp/
npm i -D -E @vue/cli-service
ionic serve

```

## Install Project:

```
nvm use 18.12.1 || nvm install 18.12.1
npm install -g npm@9.1.2
npm install -g @ionic/cli
cd ionic-app/
npm install
ionic serve
```
